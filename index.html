How to Play Adinkra Mahjong

Match identical Adinkra symbols to clear the board. The game gets harder each level with more stacked tiles!

  • Match Tiles: Click on two identical symbols to remove them
  • Green Highlight: Correct matches turn green
  • Stacked Tiles: Tiles are stacked in layers - only top tiles can be selected
  • Level Progression: Each level adds more tiles and complex layouts
  • Learn Meanings: Each symbol has cultural significance
